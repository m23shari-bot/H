🧠 المهارات المطلوبة

  • React (Hooks, Context, Redux أو Zustand)
  • CSS متقدم (Flex, Grid, Animations)
  • Responsive & Cross-Browser
  • مبادئ UI/UX عميقة
  • تحويل الأفكار لواجهات حية
  • اهتمام بالتفاصيل وجودة التنفيذ

ما نقدمه

  • مشروع تقني طموح في التعليم الرقمي

  • منتج ذو تأثير حقيقي وقيمة عالية للمستخدمين

مميزات إضافية (تفضل)
